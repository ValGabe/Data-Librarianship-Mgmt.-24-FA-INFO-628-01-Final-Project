To extract the latitude and longitude from the coordinates string, create a new column for latitude by selecting the original coordinates column and choosing "Edit column > Add column based on this column." In the dialog box that appears, enter a name for the new column, such as "Latitude," and use the GREL expression:
value.split(",")[0]
This splits the coordinate string at the comma and selects the first part as the latitude. Next, repeat this process for longitude by adding a new column titled "Longitude" and using the GREL expression:
value.split(",")[1]
This extracts the second part of the coordinate string as the longitude.
